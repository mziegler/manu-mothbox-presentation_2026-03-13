What is a Mothbox?

  1. Shines lights on a sheet: UV and visible
  2. Takes a photo every 60 seconds
  3. We use AI to identify insects in the photos
  4. Normally, you need to change the battery and memory card each night (because the light uses a lot of power)

  • Limitations
    1. Works better for large insects
    2. AI currently only goes to order level, but is improving
    3. In some cases, it will never get to species level
    4. It cannot count individuals or abundance well
    5. Biased sampling?
  • Potentials
    1. AI identification models will improve in 3-5 years
    2. Collects large volumes of presence data
    3. Longitudinal data - changes over time?
    4. Proxy measures of biodiversity?
    5. Behavioral data?

Next week: Networked deployment

  • Two mothboxes: powered with solar, connected to network with fiber optic
  • Set it up and forget it: hopefully they will require minimal maintainance, and collect data every night
  • Testing: what will break? Is it easy to maintain/fix?
  • AI for species identification will improve in the future, so we will just collect data now and analyze later
  • (But we would love help figuring out what analysis to do!)

Future: More Mothboxes and Infrastructure?

  • We are applying for grants to do a bigger monitoring version of this project
  • We can monitor insects over time, around different parts of the field station
  • The network and infrastructure will be a big investment, to get fiber and power around the whole conservancy. Once we build it, we can also use it for other sensors
  • We need to develop new methods to analyze and use this data, because the technology is so new.
